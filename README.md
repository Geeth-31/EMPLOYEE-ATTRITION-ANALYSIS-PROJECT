# Employee Attrition Analysis using Python & Tableau

## Project Overview

Employee attrition is one of the most important challenges faced by organizations because it can lead to increased recruitment costs, reduced productivity, loss of experienced employees, and lower team morale. This project analyzes employee attrition data to identify patterns associated with employee turnover and provide data-driven insights that can support better HR decision-making.

The analysis is performed using Python for data preparation and exploratory analysis, while Tableau is used to build an interactive dashboard for business users.

---

## Business Problem

Employee turnover affects both operational efficiency and business costs. Understanding the factors associated with employee attrition can help HR teams identify at-risk employee groups and develop strategies to improve employee retention.

This project aims to transform employee data into actionable business insights that support informed decision-making.

---

## Project Objectives

- Understand the employee attrition dataset.
- Perform data cleaning and preprocessing.
- Conduct exploratory data analysis (EDA).
- Analyze key HR metrics and business KPIs.
- Build an interactive Tableau dashboard.
- Generate business insights supported by data.
- Provide practical recommendations to improve employee retention.
